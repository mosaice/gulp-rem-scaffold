# gulp-rem-scaffold
## gulp rem atuo develop scaffold
*how to use?*

**clone**
```
git clone git@github.com:mosaice/gulp-rem-scaffold.git
```
**install**
```
npm install
```
**start**
```
gulp
```
listen on http://localhost:3000

### 使用gulp搭建的rem自动开发环境
*如何使用?*

```
git clone git@github.com:mosaice/gulp-rem-scaffold.git
```

```
npm install
```

```
gulp
```
